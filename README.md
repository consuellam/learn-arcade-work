A project to help me practice my Python skills while learning game development using the [Python Arcade Library](https://api.arcade.academy/en/stable/).

This is part of their [Learn Arcade Book](https://learn.arcade.academy/en/latest/index.html#). The book however is base on a previous 2.6 version of Arcade. So this is my attempt to use the Labs but use the current 3.0 version with my own spin on things.

- I setup uv to mangage my python environment and package mangagement
- I used vscode instead of pycharm as my ide.
- I chose to keep all files for each Lab in that Labs directory instead of a separent Scratch folder.
